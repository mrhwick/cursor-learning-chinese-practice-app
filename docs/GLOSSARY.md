# Glossary (quick definitions)

- **Repo**: a folder containing code and history.
- **Frontend**: the part you see in the browser (React).
- **Backend**: the server that handles requests (FastAPI).
- **API**: a set of URLs the frontend can call to get work done.
- **Server**: a program that listens for requests.
- **Port**: a number in a URL that chooses which server program you’re talking to (e.g. `:5173`).
- **Dependency**: a library this project uses; you install it with `npm` or `pip`.
- **npm**: the Node.js package installer.
- **pip**: the Python package installer.
- **venv**: a “virtual environment” that keeps Python packages for this project separate from your system.
- **JSON**: a common format for data sent between frontend and backend.
- **Cookie**: a small piece of data stored by the browser; used here for login sessions.
